# Recommended Stable Dependency Ranges

These ranges are validated for the Cost Forge 2 sandbox.

- streamlit >=1.45,<1.46
- pandas >=2.2,<2.3
- openpyxl >=3.1,<3.2
- numpy >=1.26,<2.0
- plotly >=5.22,<5.23
- reportlab >=4.2,<4.3
- python-docx >=1.1,<1.2
- pytest >=8.2,<8.3
- ruff >=0.5,<0.6

Purpose:
- Reduce deployment drift
- Improve Streamlit stability
- Avoid unexpected major-version breaks
